Template Methods [Gamma1995] for graph algorithms.

The template methods in this package were designed after analyzing the generic graph algorithms in the Boost Graph Library (BGL) [Siek2001]. Note that the BGL and JavaUtils implement generic graph algorithms somewhat differently:

  • JavaUtils define Template Methods rather than Visitors [Gamma1995] like BGL. Template methods are comparatively more convenient to use in Java than in C++.
  • The overridable event point methods defined by JavaUtils template methods are slightly different from the member functions defined by BGL. In particular, the BFS Visitor of BGL and the {@link javautils.graph.templates.BfsTemplate} have different semantics for finish_vertex() and finishNode(), respectively. The reason for these differences is that the current semantics of JavaUtils seem simpler to the author.
