JavaUtils

JavaUtils is a library of Java utility classes and packages. Current main features include:

  • Framework for functional objects.
  • Utilities for manipulating collections, including many sequence algorithms implemented as higher-order methods.
  • Utilities for manipulating graphs, including template methods for graph algorithms.
  • Dispenser framework.
  • Utilities for dealing with JDBC.

See the JavaDoc overview for further information.
